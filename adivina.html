Bienvenido al Juego de Adivinanza

Puedes adivinar el número secreto entre 1 y 100?

Este es un juego de adivinanza simple en el que el usuario debe adivinar un número secreto generado aleatoriamente. Puedes expandir este juego o tomarlo como inspiración para crear otros juegos más complejos. Si estás interesado en crear un juego más elaborado, te recomiendo aprender JavaScript y utilizar una biblioteca de desarrollo de juegos como Phaser o Three.js para proyectos 2D o 3D, respectivamente. Estas bibliotecas te proporcionarán herramientas y funciones más avanzadas para crear juegos interactivos y complejos en la web.
